Characterization tips

1 April 2024
0 comments

Tooltips for features help you display the necessary explanations for product parameters to help customers make the right choice.

For example, for the Size characteristic, you can set a tooltip that contains a dimensional grid.

 

A tooltip can be displayed on a website in two formats: as a text link (as in the example above) or as an information icon.

Displaying tooltips in the form of an icon


You can customize the format of the tooltip display in the section of the admin panel Settings → General settings → Product catalog.

 

The tooltips are displayed on the product page and in the mini-card in the catalog.

How to create a feature tip
To create a new tooltip:

Go to Settings → Feature Hints and click Add.

Make the necessary settings:

Name - will be displayed on the product page if the tooltip is in the format of a text link;

Select the Template and Parameter (characteristic) for which this tooltip is created;

Set the Brands for which the tooltip will be displayed by selecting the appropriate checkboxes;

Text - in this field, you specify the content of the tooltip. Here you can place text blocks, images, links, and format them using the visual editor.

Display in modal window - if the checkbox is selected, the tooltip will be displayed in a separate window that pops up on top of the page.

3. To activate the tooltip, select the Enabled checkbox and click Save.
 

 

You can set the prompt separately for each language version of the site.
